Xdebug
======
http://xdebug.org/

Tailored Installation Instructions via http://www.xdebug.org/find-binary.php

Summary
Xdebug installed: no
Server API: Apache 2.0 Handler
Windows: yes - Compiler: MS VC6 - Architecture: x86
Zend Server: no
PHP Version: 5.3.5
Zend API nr: 220090626
PHP API nr: 20090626
Debug Build: no
Thread Safe Build: yes
Configuration File Path: C:\WINDOWS
Configuration File: C:\#Braiins\UniServer\usr\local\php\php.ini
Extensions directory: C:\#Braiins\UniServer\usr\local\php\extensions
Instructions

    Download php_xdebug-2.1.2-5.3-vc6.dll
    Move the downloaded file to C:\#Braiins\UniServer\usr\local\php\extensions
    Edit C:\#Braiins\UniServer\usr\local\php\php.ini and add the line
    zend_extension = C:\#Braiins\UniServer\usr\local\php\extensions\php_xdebug-2.1.2-5.3-vc6.dll
    Restart the webserver
