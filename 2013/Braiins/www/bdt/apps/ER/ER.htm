Reset Entity

Click the row of an Entity to Reset it.

  • Resetting an entity means deleting ALL accounting data for the entity. It is equivalent to deleting the entity and setting it up again with the same entity information. By comparison, if you delete an entity it cannot be set up again with the original entity reference.
  • Only entities with data are listed here.
  • If you click to select an entity, a further confiirmation step occurs before the data is reset.
